#  Sales Performance Analysis & Dashboard

##  Project Overview
This project analyzes 51,000+ global retail transactions to evaluate sales performance, profitability trends, and regional growth patterns.  
The objective was to extract meaningful business insights and build an interactive dashboard to support data-driven decision-making.



##  Tech Stack
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Microsoft Power BI
- CSV Dataset


##  Dataset Information
- 51,000+ transaction records
- Features include:
  - Order Date
  - Region
  - Category
  - Sales
  - Profit
  - Discount
  - Shipping Cost
  - Customer Details



##  Project Workflow

### 1️ Data Cleaning & Preprocessing
- Converted date columns to datetime format
- Handled datatype inconsistencies
- Created new features (Month, Year)
- Verified numeric fields for analysis

### 2️ Exploratory Data Analysis (EDA)
- Calculated Total Sales and Total Profit
- Analyzed Sales by Region
- Evaluated Profit by Category
- Identified Top 10 Customers
- Examined Monthly Sales Trends

### 3️ Business Analysis
- Studied impact of discount on profitability
- Identified loss-making products
- Compared profit margins across categories
- Evaluated yearly sales growth trends

### 4️ Dashboard Development
Built an interactive dashboard using Microsoft Power BI including:
- Total Sales KPI
- Total Profit KPI
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Top Customers
- Interactive filters (Region, Category, Year)



##  Key Insights
- Certain regions contribute significantly higher revenue.
- Higher discount levels reduce average profit margins.
- A small percentage of customers generate a major share of revenue.
- Some products consistently result in financial losses.
- Sales show seasonal growth patterns.


##  Conclusion
This project demonstrates end-to-end data analysis skills including data cleaning, exploratory analysis, business insight generation, and dashboard creation.  
It highlights the ability to transform raw transactional data into actionable business intelligence.
